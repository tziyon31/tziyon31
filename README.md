# Tziyon Bublil — DevOps / Cloud Engineer

DevOps and cloud engineer focused on AWS architecture, CI/CD automation, Kubernetes, release governance, and production-oriented infrastructure.

I work on systems that connect infrastructure design with delivery workflows: environment strategy, artifact promotion, deployment automation, observability, cost awareness, and operational reliability.

Current focus areas include AWS ECS/Fargate, Kubernetes, Terraform, GitHub Actions, Jenkins, GitOps, monitoring stacks, and secure multi-environment delivery.
